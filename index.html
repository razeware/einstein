Android Table of Elements

Understanding what to learn and in what order to learn Android development can be confusing. So — we created the Android Table of Elements to help students on their path to be a professional Android developer.

Android tutorials have been organised into elements. Each element is part of a broader macro topic. Learn elements to become an expert in these macro topics and gradually build your skills from the top of the table to the bottom.
